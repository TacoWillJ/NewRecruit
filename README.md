# No Figure Left Behind — WH40K 10th Edition [NFLB]

A full Warhammer 40,000 10th Edition game system for New Recruit/BattleScribe,
based on BSData's wh40k-10e data with NFLB homebrew units added alongside official content.

## How to use

In New Recruit (web or iOS):

1. Menu → My Games → **+** → enter `https://github.com/TacoWillJ/NewRecruit`
2. Select **Warhammer 40,000 10th Edition [NFLB]** from your games list
3. Build your list normally — all official units are present, NFLB homebrew units appear alongside them marked with the **NFLB** source badge

## What's included

All 46 official faction catalogues from BSData wh40k-10e, plus:

| Faction | NFLB Units |
|---|---|
| Astra Militarum | Infantry Platoon v3 (Platoon Command Squad + Infantry Squads) |

More factions coming as homebrew cards are built.

## Design philosophy

- All official units included — this replaces the BSData source, not supplements it
- NFLB homebrew units sit alongside official units exactly like Legends units do
- Every homebrew unit is marked with the **No Figure Left Behind** source badge
- Units without a current GW miniature get a homebrew 10th Edition datasheet
- Flat per-unit point costs, no weapon ladders
- Regiment-agnostic where possible

## Keeping up to date

This repo tracks BSData's wh40k-10e. When GW releases balance updates, the
affected faction catalogues are pulled from BSData and re-released here with
NFLB additions intact.
