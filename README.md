# No Figure Left Behind — New Recruit Catalogues

Homebrew BattleScribe/New Recruit supplement catalogues for discontinued Warhammer 40,000 units.

## How to use

These are **supplement catalogues**, not replacements. In New Recruit:

1. Build your army using the official faction catalogue as normal
2. Add this repo as a second data source (Menu → My Games → + → GitHub/URL)
3. NFLB homebrew units appear alongside all standard units, tagged **[NFLB]**

## Available catalogues

| File | Faction | Contents |
|---|---|---|
| `NFLB - Astra Militarum.cat` | Astra Militarum | Infantry Platoon v3 |

## Design philosophy

- Units without a current GW miniature get a homebrew 10th Edition datasheet
- All units are clearly marked with the **No Figure Left Behind** source badge
- Flat per-unit point costs, no weapon ladders
- Regiment-agnostic where possible
